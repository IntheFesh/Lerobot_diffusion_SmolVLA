# CV Bullet Points

- Built a reproducible imitation-learning research stack on LeRobot + LIBERO,
  including baseline replication, experiment automation scripts, and cloud-ready
  multi-GPU training configurations.
- Designed and implemented **PhaseQFlow**, a phase-aware and quality-weighted
  generative action policy that improves long-horizon manipulation training
  stability and data efficiency.
- Implemented operational tooling for dataset diagnostics, checkpoint export,
  evaluation, and latency benchmarking to support research iteration and model
  deployment workflows.
- Extended the project with SmolVLA LoRA fine-tuning paths and modular
  architecture choices that support future VLA and online RL integration.
